🔐 Private cross-device sync · Google Drive

        
Homebase encrypts your memory, tasks, AI preferences, rolling usage ledger, and—when enabled—provider credentials on this device before they touch Google. Drive only stores ciphertext. Your passphrase stays in this browser session and is never uploaded.

        

          
        

        

          
          
          
        

        

          
          
          
        

        

          Keep this device in syncPushes local changes and checks for newer mobile/desktop changes while Homebase is open
          

        

        

          Sync API keys and private connectorsEncrypted before upload. Turn off to sync only non-secret preferences, memory, tasks, and usage.
          

        

        

        
Setup: Google Cloud → OAuth client (Web) → authorized JS origin https://wghtkbpxwx-a11y.github.io → paste Client ID under Connectors → Google Drive.
